## Permissions requises

Albion Online Objectives (AOO) a besoin des permissions suivantes pour fonctionner correctement :

| Permission | Raison |
|-----------|--------|
| **Voir les salons** | Accéder aux salons où les commandes sont utilisées |
| **Envoyer des messages** | Publier les listes d'objectifs et les réponses |
| **Voir les anciens messages** | Retrouver les anciens messages du bot pour les mettre à jour |
| **Gérer les messages** | Supprimer les messages d'objectifs obsolètes lors du rafraîchissement |
| **Utiliser les commandes slash** | Enregistrer et répondre aux commandes `/add`, `/list`, `/remove` |

## Aucune configuration nécessaire

Albion Online Objectives fonctionne immédiatement après l'installation — il n'y a aucune commande de configuration. N'importe quel membre de votre serveur peut commencer à rapporter des objectifs avec la commande `/add`.
