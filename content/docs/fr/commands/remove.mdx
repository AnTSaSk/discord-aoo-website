La commande `/remove` supprime un objectif rapporté de la liste active de votre serveur.

## Syntaxe

```
/remove index:<number>
```

## Paramètres

| Paramètre | Type | Obligatoire | Description |
|-----------|------|----------|-------------|
| `index` | Nombre | Oui | Le numéro d'index de l'objectif à supprimer (affiché dans le résultat de [`/list`](/docs/commands/list/)) |

## Permissions

Vous pouvez supprimer un objectif si vous remplissez **l'une** de ces conditions :

| Qui | Condition |
|-----|-----------|
| **Auteur du rapport** | Vous avez ajouté l'objectif avec `/add` |
| **Gérer les messages** | Vous disposez de la permission Gérer les messages |
| **Administrateur** | Vous disposez de la permission Administrateur |

Si vous n'avez pas la permission requise, le bot répond avec un message d'erreur (visible uniquement par vous).

## Comment trouver l'index

1. Exécutez [`/list`](/docs/commands/list/) pour voir tous les objectifs actifs
2. Chaque entrée commence par `#N` — c'est le numéro d'index
3. Utilisez ce numéro avec `/remove`

## Comportement

### Rafraîchissement du message

Après une suppression réussie, le bot rafraîchit automatiquement le message de la liste des objectifs pour refléter le changement. Les numéros d'index sont recalculés.

### Temps de recharge

Cette commande a un **temps de recharge de 2 secondes** par utilisateur.

## Commandes associées

- [`/list`](/docs/commands/list/) — Voir les objectifs actifs et leurs numéros d'index
- [`/add`](/docs/commands/add/) — Rapporter un nouvel objectif
