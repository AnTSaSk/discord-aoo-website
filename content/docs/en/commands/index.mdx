All Albion Online Objectives (AOO) interactions use Discord slash commands. Type `/` in any channel where the bot has access to see available commands.

<Callout variant="info">Commands have cooldowns to prevent spam — 2 seconds for /add and /remove, 5 seconds for /list.</Callout>

## Commands

| Command | Description | Cooldown |
|---------|-------------|----------|
| [`/add`](/docs/commands/add/) | Report a new Black Zone objective | 2 seconds |
| [`/list`](/docs/commands/list/) | View all active objectives for your server | 5 seconds |
| [`/remove`](/docs/commands/remove/) | Delete a reported objective | 2 seconds |

## Quick Start

New to Albion Online Objectives? Follow the [Report Your First Objective](/docs/getting-started/first-objective/) tutorial to learn the basic workflow:

1. **Report** an objective you find in the Black Zone with [`/add`](/docs/commands/add/)
2. **View** all active objectives for your guild with [`/list`](/docs/commands/list/)
3. **Remove** an expired or incorrect entry with [`/remove`](/docs/commands/remove/)

## Permissions

<Badge variant="green">No permissions required</Badge> All three commands are available to every member of your server. The only exception is `/remove`, which requires one of the following to delete another player's report:

- **Administrator** permission
- **Manage Messages** permission
- Being the original reporter

See each command's dedicated page for full parameter details and examples.
