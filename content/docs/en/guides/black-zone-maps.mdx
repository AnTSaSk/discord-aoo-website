Albion Online Objectives (AOO) supports **276 Black Zone maps** for the [`/add`](/docs/commands/add/) command. The `map` parameter uses Discord autocomplete — start typing any part of the map name (beginning, middle, or end) and matching results will appear.

## Using Map Autocomplete

When using the [`/add`](/docs/commands/add/) command, start typing any part of the map name:

- **Beginning**: Type `Murder` to find Murdergulch maps
- **Middle**: Type `Cross` to find maps containing "Cross" (Dryvein Cross, Everwinter Crossing, etc.)
- **End**: Type `Ravine` to find all maps ending in Ravine

The autocomplete will show up to 25 matching results at a time.
