## Required Permissions

Albion Online Objectives (AOO) needs the following permissions to work properly:

| Permission | Reason |
|-----------|--------|
| **View Channel** | Access channels where commands are used |
| **Send Messages** | Post objective lists and responses |
| **Read Message History** | Find previous bot messages to update them |
| **Manage Messages** | Delete outdated objective messages during refresh |
| **Use Slash Commands** | Register and respond to `/add`, `/list`, `/remove` |

## No Configuration Needed

Albion Online Objectives works immediately after installation — there is no setup command. Any member of your server can start reporting objectives right away using the `/add` command.
